# 📊 Data Analytics & Visualization Projects

This repository contains a collection of end-to-end data analysis and visualization projects spanning business intelligence, healthcare analytics, election data insights, and machine learning applications.

The projects demonstrate practical applications of EDA, dashboarding, statistical analysis, and classification modeling using tools such as Python, Tableau, and Power BI.

---

## 🚀 Projects Overview

### 1. 📈 PoBI & Tableau Project
- Focus: Business Intelligence & Data Visualization  
- Tools: Tableau, Power BI  
- Developed interactive dashboards for business insights and KPI tracking.

### 2. ❤️ Cardio DB Analysis
- Domain: Healthcare Analytics  
- Tools: Python (Pandas, NumPy, Matplotlib, Seaborn)  
- Performed EDA to identify key cardiovascular risk factors.

### 3. 🗳️ Election DB Analysis
- Domain: Political Data Analysis  
- Tools: Python, Tableau  
- Analyzed voter trends and regional patterns.

### 4. 🍔 Food Delivery Data Analysis
- Domain: Logistics & Business Analytics  
- Tools: Python  
- Evaluated delivery efficiency, cost, and customer behavior.

### 5. 🧠 Mental Health EDA + Diagnosis Classification
- Notebook: mental-health-eda-diagnosis-classification.ipynb  
- Built classification models and performed detailed EDA.

### 6. 📓 Kaggle Notebook – Mental Health Analysis
- Extended version with feature engineering and model experimentation.
